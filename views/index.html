TimeStamp MicroService

By using this A.P.I/MicroService you can get unix time of your input Date or can convert Unix time to U.T.C

How to Use

Just put this U.R.L - "https://Unix2Utc/api/timestamp/date" Replacing last word date with real date in YY-MM-DD format (all numeric)

Example Usage

https://Unix2Utc/api/timestamp/2021-04-08
